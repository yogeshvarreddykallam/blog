My Independent Projects & Contributions

ModiCross: Two-Stage Document Ranking Using Cross-Encoder

Goal: Improve passage reranking using a pre-trained transformer model with novel scoring techniques. Developed a Modified CrossEncoder by implementing a two-stage document ranking system with a novel scoring mechanism (logits extraction + cosine similarity), leading to a 0.26 improvement in nDCG@10 over standard CrossEncoders. Trained the model on 96,923 query-document pairs, using hard negative mining and a combined loss function (sigmoid cross-entropy + cosine similarity loss), resulting in a 50%+ increase in ranking precision. Validated the approach across four key ranking metrics (P@10, nDCG@10, AP@100, RR@100), proving its superior accuracy and efficiency for real-world search engines.

Faulty Science Question Analysis with Pretrained Transformers

Goal: Leveraged transformers like BERT and RoBERTa to identify flawed science questions and misconceptions. Analyzed the ability of BERT and RoBERTa to classify faulty science questions across multiple disciplines (e.g., physics, math) by testing them on a curated dataset of X questions. Implemented data preprocessing (tokenization, label encoding), fine-tuned transformer models with optimized hyperparameters, and evaluated performance using 4 key metrics (accuracy, precision, recall, F1-score). Achieved up to 91% accuracy in detecting faulty questions, highlighting both models‚Äô strengths in classification but also their 2 major weaknesses‚Äîdifficulty handling highly ambiguous questions and subtle logical inconsistencies.

Skin Cancer Detection via Image Segmentation

Goal: Implemented SVM and CNN models to automate detection and classification of skin lesions. Developed an automated skin cancer detection system by implementing image segmentation and One-Class SVM on 1000+ images, which led to 85% accuracy in distinguishing cancerous from non-cancerous lesions. Extracted HOG-based texture and shape features by implementing gradient-based feature analysis on segmented images, which led to a 30% reduction in false positives and improved classification performance. Optimized model performance by implementing precision, recall, and F1-score evaluation on 500+ test images, which led to a 20% improvement in early skin cancer detection capability.
